# fugle-backend-pretest

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Tech Stack

[Nest.js](https://github.com/nestjs/nest), [TypeScript](https://www.typescriptlang.org/), [WebSocket](https://www.npmjs.com/package/ws), [Redis](https://redis.io/) and [Deciaml.js](https://github.com/MikeMcl/decimal.js#readme)

## Requirements

- [X] Part 1: HTTP API
- [X] Part 2: Rate Limiting
- [X] Part 3: WebSocket API

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
